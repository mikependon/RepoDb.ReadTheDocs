Supports
========

Below are the currently supported versions of .Net and SQL Server.

- .NET Framework: Yes (>= 4.5)
- .NET Core: Yes (>= v2.1)
- .NET Standard Library: Yes (>= 2.0)
- SqlServer: YES
- MySql: YES
- SqLite: YES
- PostgreSql: YES (>= v12.1)
- Other RDBMS: can be extended
- NoSql Databases: No
